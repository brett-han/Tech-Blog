### Mac Keyboard Shortcuts

#### App Control
- **Open app list**  
    1. Open **System Settings**.
    2. Go to **Keyboard**.
    3. Click **Keyboard Shortcuts**.
    4. Select **Launchpad & Dock**.
    5. Find **Show Launchpad**.
    6. Set your preferred shortcut (for example: `Cmd + Q`).
    
  `Cmd + Q`

- **Close window**  
  `Cmd + W`

#### Finder
- **Open Finder search window**  
  `Cmd + Option + Space`

#### Dock
- **Show / Hide Dock**  
  `Cmd + Option + D`

#### Input Method
- **Switch between Chinese and English (Pinyin input)**  
  Press `Caps Lock`
